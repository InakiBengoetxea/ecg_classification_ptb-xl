# Signal + Metadata Interpretability in Arrhythmia Classification from 12-lead ECG: Quantifying ECG-morphology vs Patient-feature Contributions
Repository associated to "Signal + Metadata Interpretability in Arrhythmia Classification from 12-lead ECG: Quantifying ECG-morphology vs Patient-feature Contributions". Classification of ECG signals with xresnet1d101 using PTB-XL dataset.

The dataset can be accessed through this link: https://physionet.org/content/ptb-xl/1.0.3/
